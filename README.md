# Customer Retention Analysis

## Project Overview

Customer Retention Analysis is an end-to-end Data Analytics project that analyzes customer purchasing behavior, Customer Lifetime Value (CLV), repeat purchasing patterns, and overall business performance using SQL, Python, Excel, and Power BI. The project demonstrates the complete analytics workflow—from data preparation and cleaning to SQL analysis, exploratory data analysis, business reporting, and interactive dashboard development.

---

# Key Findings

- 💰 **Total Revenue:** ₹115 Million
- 🛒 **Total Orders:** 4,780+
- 👥 **Total Customers:** 987
- 🔄 **Repeat Customers:** 951
- 👤 **One-Time Customers:** 36
- 📈 **Repeat Customer Rate:** 95.10%
- 💎 **Highest Customer Lifetime Value (CLV):** ₹409K
- 📊 **Average Customer Lifetime Value:** ₹114.94K
- 💵 **Average Order Value:** ₹24.05K
- 📦 **Total Quantity Sold:** 44,965 Units
- 📅 **Peak Revenue Month:** April (~₹12.8M)
- 📉 **Lowest Revenue Month:** September (~₹7.3M)
- 🎯 A small group of high-value customers contributes a significant portion of overall revenue.
- 📌 Customer activity declines during the second half of the year, indicating opportunities to improve customer retention.
